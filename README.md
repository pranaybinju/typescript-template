# react-native-web-typescript-template

The template consist React Native and React Web with Typescript and Webpack.

Usage
---

To rename app run
```
yarn rename app_name
```

To clean app run
```
yarn clean
```

Run it on browser
```
yarn web
```

Run it on android
```
yarn android
```

Run it on ios
```
yarn ios
```

To start metro bundler run
```
yarn start
```
